# Website Checklist — R.AI Landing

Danh sách các hạng mục cần bổ sung/cải thiện. Cập nhật trạng thái khi hoàn thành.

---

## 🔴 Ưu tiên cao

- [ ] **Case Studies / Portfolio** — Tạo trang `/case-studies` với ít nhất 2-3 dự án đã triển khai (bài toán → giải pháp → kết quả đo được)
- [ ] **Hình ảnh & Visual** — Bổ sung screenshot sản phẩm, diagram kiến trúc, video demo cho trang Technology và Solutions
- [ ] **Form backend thực** — Thay mailto bằng API endpoint (Resend, Formspree, hoặc custom API) để lưu lead vào database
- [ ] **Blog content thực** — Viết ít nhất 3-5 bài blog chất lượng để SEO hoạt động

## 🟡 Ưu tiên trung bình

- [ ] **Client logos / Đối tác** — Thêm section logo khách hàng/đối tác trên trang chủ
- [ ] **Testimonials** — Thu thập và hiển thị feedback từ khách hàng thực
- [ ] **Analytics** — Tích hợp Google Analytics 4 hoặc Plausible/Umami
- [ ] **CTA cuối mỗi trang** — Thêm block CTA ở cuối trang Solutions, Technology, About
- [ ] **Đội ngũ đầy đủ** — Bổ sung thêm thành viên hoặc mô tả quy mô team trong trang About
- [ ] **Pricing / Mô hình hợp tác** — Mô tả các hình thức hợp tác (Consulting, Platform, Custom Dev)
- [ ] **Open Graph Image** — Kiểm tra và tối ưu OG image cho từng trang

## 🟢 Ưu tiên thấp

- [ ] **Careers / Tuyển dụng** — Tạo trang `/careers` khi có nhu cầu tuyển
- [ ] **Multi-language (EN)** — Phiên bản tiếng Anh khi mở rộng thị trường quốc tế
- [ ] **Loading states** — Thêm skeleton/loading UI cho các trang
- [ ] **PWA icons** — Bộ favicon đầy đủ cho mobile bookmark và PWA
- [ ] **404 page** — Review và cải thiện nội dung trang not-found
- [ ] **Sitemap động cho blog** — Đảm bảo sitemap tự cập nhật khi thêm bài blog mới

## 🔧 Kỹ thuật

- [ ] **Performance audit** — Chạy Lighthouse, tối ưu LCP/CLS
- [ ] **Accessibility** — Kiểm tra contrast, aria-labels, keyboard navigation
- [ ] **Error monitoring** — Tích hợp Sentry hoặc tương đương
- [ ] **Backup contact method** — Thêm Zalo/Messenger widget hoặc link chat
